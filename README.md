# ResQMeal

An AI-powered Food Waste Management Platform that connects restaurants with nearby NGOs to reduce food wastage.

## Features
- Restaurant Dashboard
- NGO Dashboard
- Delivery Partner Module
- AI Food Safety Prediction
- Smart NGO Recommendation
- Route Optimization
- Admin Dashboard

## Tech Stack
- React.js
- Flask
- MySQL
- Scikit-learn
- Leaflet / Google Maps

🚧 Currently under development.
